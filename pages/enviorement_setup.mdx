### GFortran-ın Quraşdırılması

GFortran, GNU Compiler Collection (GCC)-nin bir hissəsi olan GNU Fortran kompilyatorudur. Onun quraşdırılma prosesi əməliyyat sisteminizdən asılı olaraq fərqlənir.

---

### **Windows Üçün Installer Seçimləri**

Windows üzərində GFortran-ı quraşdırmağın installer-dən istifadə edəcyik

   - **Equation.com:** GCC 12.1 versiyası üçün 32-bit və 64-bit x86 faylları təqdim edir.
   - **TDM GCC:** GCC 10.3 versiyası üçün 32-bit və 64-bit faylları təmin edir.
   - **MinGW-w64:** GCC 12.2 versiyası üçün 64-bit x86 faylı təqdim edir.

   Yuxarıdakı mənbələrdən birini seçib, installer-i yükləyin və quraşdırma prosesi üçün təlimatları izləyin.

---

### **Linux Üzərində Quraşdırma**

#### **Debian əsaslı Dağıtımlar (Debian, Ubuntu, Mint və s.)**

1. **GFortran-ın Quraşdırılıb-quraşdırılmadığını Yoxlayın:**
   Terminalı açaraq aşağıdakı əmri daxil edin:
   ```
   which gfortran
   ```
   Əgər heç bir nəticə qaytarılmırsa, GFortran quraşdırılmayıb.

2. **GFortran-ı Quraşdırın:**
   GFortran kompilyatorunu quraşdırmaq üçün aşağıdakı əmrdən istifadə edin:
   ```
   sudo apt install gfortran
   ```
   Quraşdırılmış versiyanı yoxlamaq üçün:
   ```
   gfortran --version
   ```

3. **Xüsusi Versiyaları Quraşdırın:**
   Məsələn, 8-ci versiyanı quraşdırmaq üçün:
   ```
   sudo apt install gfortran-8
   ```
   Əgər Ubuntu-nun köhnə versiyasında yeni versiyaları quraşdırmaq istəyirsinizsə, aşağıdakı əmrləri daxil edin:
   ```
   sudo add-apt-repository ppa:ubuntu-toolchain-r/test
   sudo apt update
   sudo apt install gfortran-10
   ```

4. **Versiyalar Arasında Keçid:**
   `update-alternatives` əmri vasitəsilə quraşdırılmış versiyalar arasında keçid edə bilərsiniz. Bununla bağlı onlayn dərsliklər mövcuddur.

#### **RPM əsaslı Dağıtımlar (Red Hat, Fedora, CentOS, openSUSE)**

1. **GFortran-ı Quraşdırın:**
   Red Hat və CentOS-da `yum` istifadə edin:
   ```
   sudo yum install gcc-gfortran
   ```
   Fedora və RHEL (8 və yuxarı) versiyalarında `dnf` istifadə edin:
   ```
   sudo dnf install gcc-gfortran
   ```

#### **Arch əsaslı Dağıtımlar (Arch Linux, Manjaro, EndeavourOS)**

1. **GFortran-ı Quraşdırın:**
   Arch əsaslı sistemlərdə `pacman` istifadə edin:
   ```
   sudo pacman -S gcc-fortran
   ```

### **macOS Üzərində Quraşdırma**

#### **Xcode İstifadə Edərək:**

1. **Xcode Alətlərini Quraşdırın:**
   Əgər Xcode quraşdırılıbsa, terminalı açın və aşağıdakı əmri daxil edin:
   ```
   xcode-select --install
   ```

#### **Homebrew İstifadə Edərək:**

1. **GCC-ni Quraşdırın:**
   Homebrew istifadə edirsinizsə, GCC-ni (hansı ki GFortran-ı da daxil edir) bu əmr vasitəsilə quraşdırın:
   ```
   brew install gcc
   ```

#### **Fink və ya MacPorts İstifadə Edərək:**

1. **Fink:** GNU GCC paketini axtarın:
   ```
   fink search gcc
   ```
   GCC-nin bir versiyasını quraşdırın:
   ```
   sudo port install gcc10
   ```

2. **MacPorts:** MacPorts ilə istədiyiniz GCC versiyasını quraşdırın:
   ```
   sudo port install gcc10
   ```

#### **Birbaşa İkilik Fayllar ilə Quraşdırma:**

1. **FXcoudert’s GFortran for macOS:**
   GFortran üçün əvvəlcədən hazırlanmış ikilik faylları [fxcoudert GitHub deposu](https://github.com/fxcoudert/gfortran-for-macOS) vasitəsilə yükləyə bilərsiniz.

---

### **OpenBSD Üzərində Quraşdırma**

OpenBSD-də GFortran icra faylı `egfortran` adlanır. Quraşdırmaq üçün aşağıdakı əmri işlədin:
```
pkg_add g95
```
GFortran versiyasını yoxlamaq üçün:
```
egfortran -v
```

---

### **OpenCoarrays (Fortran üçün Paralel Proqramlaşdırma)**

OpenCoarrays, GFortran-ın Fortran 2018 paralel proqramlaşdırma xüsusiyyətlərindən istifadə etməsinə imkan verən açıq mənbəli bir layihədir. Bu, ayrıca kompilyator deyil, GFortran ilə birlikdə paralel icra edilə bilən proqramlar yaratmaq üçün işləyir.

1. **Paralel Kod İcra Etmək:**
   OpenCoarrays ilə paralel Fortran proqramlarını icra etmək üçün aşağıdakı əmr istifadə olunur:
   ```
   cafrun -n <şəkillərin_sayı> <icra_faylı_adı>
   ```

2. **Quraşdırma:**
   OpenCoarrays-ı quraşdırmaq üçün [OpenCoarrays rəsmi quraşdırma təlimatı](https://github.com/sourceryinstitute/opencoarrays)-na baxın.

---

### **Windows-da Yerli Quraşdırma**

Təəssüf ki, Windows-da GFortran-ın yerli quraşdırılması mümkün deyil. Onu yalnız **Windows Subsystem for Linux (WSL)** vasitəsilə quraşdırmaq olar. Bu, Linux-a bənzər bir mühit təmin etdiyi üçün inkişaf alətləri, o cümlədən GFortran ilə tam uyğunluğu təmin edir.
