# Fortran Alətləri və Kitabxanaları

Fortran alətləri və kitabxanaları müxtəlifdir. Bəziləri pulsuz, bəziləri isə ödənişli xidmətlərdir.

Aşağıda bəzi pulsuz kitabxanalar verilmişdir:

- RANDLIB, təsadüfi rəqəm və statistik paylama generatorları
- BLAS
- EISPACK
- GAMS–NIST Mövcud Rəqəmli Proqram Təminatı Bələdçisi
- NIST-dən bəzi statistik və digər rutinlər
- LAPACK
- LINPACK
- MINPACK
- MUDPACK
- NCAR Riyazi Kitabxanası
- Netlib-in riyazi proqram təminatı, məqalələr və verilənlər bazası kolleksiyası
- ODEPACK
- ODERPACK, sıralama və tərtibat üçün rutinlər dəstəsi
- Expokit, matrisa eksponentlərini hesablamaq üçün
- SLATEC
- SPECFUN
- STARPAC
- StatLib statistik kitabxanası
- TOMS
- Sıralama və birləşdirmə sətirləri

Aşağıdakı kitabxanalar pulsuz deyil:

- NAG Fortran riyazi kitabxanası
- Visual Numerics IMSL kitabxanası
- Numerical Recipes
